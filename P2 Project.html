1. Introduction¶

We shall be looking at data from several Brazilian hospitals, who have collected data around patient appointments. The data looks at various patient factors and whether or not the appointment was attended (or was there a 'no-show'). We shall specifically investigate whether the time between scheduling and the actual appointment affects a patients chance of not attending, whether gender and other binary factors have an affect and lastly, whether a patient who has missed an appointment before, is likely to do so again.

Looking at the results, it seems that an increase in time between scheduling and attendance might affect a patient's chance of not showing up, as the mean for a no-show is almost double that of the shows. To see if this is significant and could represent the entire population, we need to find the t-test statistic for an independent sample.

Given that this t value is much higher than the critical t value for a single tail test with alpha = 0.05 (-1.645), we can assume that it is likely that an increase in time between scheduled day and appointment day may contribute to a patient not attending an appointment. To see how much it may influence it, we look at the coefficient of determinations.

3.2.3 Additional Analysis¶

Given that there are several columns with similar binary states (ie., the column entries can only be one of two entries - for example the 'Scholarship' column), it would make sense to use the gender analysis to write a function that can quickly check whether any of these affect the chance of a no-show. It seems unlikely that whether someone had diabetes or not might influence their chance of attending an appointment but if it can be done quickly it might be worth checking.

In [16]:
def statistically_significant(binary_percent_array):
    """calculates the two tailed t-test statistic for an independent set of data with alpha=0.05
    binary_percent_array should be a 4x4 pandas dataframe with rows 'No-show' and 'Show', with percentages for binary data"""
    
    #finds the means for the show and no-show data
    mean_no_show = binary_percent_array.mean().iloc[0]
    mean_show = binary_percent_array.mean().iloc[1]
    #finds the sample sizes for show and no-show data
    n_no_show = binary_percent_array.count().iloc[0]
    n_show = binary_percent_array.count().iloc[1]
    #finds the standard deviation for the show and no-show data
    stdev_no_show = binary_percent_array.std().iloc[0]
    stdev_show = binary_percent_array.std().iloc[1]
    # calculate standard error
    SEM = math.sqrt((stdev_no_show*stdev_no_show/n_no_show)+(stdev_show*stdev_show/n_show))
    # calculates the t statistic
    t_statistic = (mean_no_show - mean_show)/SEM
    #t_critical for two-tailed t-test with alpha = 0.05 and df =  inf
    t_crit = 1.960
    
    if t_statistic > t_crit:
        print('The difference between No-show and Show data is statistically significant - could be causation')
        return True
    else:
        print('The difference between No-show and Show data is NOT statistically significant - causation NOT likely')
        return False

def binary_input_on_no_show(df, binary_label, inputa, inputb):
    """a function that takes the clean dataframe (df), a specified column name (binary_label) of a binary entry column
    and the entries that column could take (inputa and inputb)."""
    #First to extract the relevant columns from the dataframe
    a_no_show = len(df[(df[binary_label] == inputa) & (df['No-show'] == 'No')])
    a_show = len(df[(df[binary_label] == inputa) & (df['No-show'] == 'Yes')])
    b_no_show = len(df[(df[binary_label] == inputb) & (df['No-show'] == 'No')])
    b_show = len(df[(df[binary_label] == inputb) & (df['No-show'] == 'Yes')])
    # Then create a new dataframe
    input_ = {inputa: pd.Series([a_no_show, a_show], index = ['No-show','Show']), inputb: pd.Series([b_no_show, b_show],index = ['No-show','Show'])}
    input_df = pd.DataFrame(input_)
    
    # test for statistical significance
    test = statistically_significant(input_df)
    
    # if the test could be significant, print a graph
    if test == True:
        # add columns that are percentages
        input_df['a_percent']=input_df[inputa]/(input_df.sum(axis = 1))*100
        input_df['b_percent']=input_df[inputb]/(input_df.sum(axis = 1))*100
        
        #make column names that relate to input
        input_names = pd.Series([inputa, inputb], index = ['a','b'])
        input_names2 = input_names.replace([0,1],['No','Has'])
        input_names3 = input_names2 + ' ' + binary_label

def likelihood_repeat(df, num):
    """calculates the probability of someone not showing up for an appointment,
    given the number of appointments they have not shown up for previously
    df - pandas DataFrame that contains frequency of no shows and count of these, labelled 'Freq_no_show' and 'Count' respectively
    num - int indicating the number of previous no shows"""
    num_no_show = df[df['Freq_no_show'] > num]['Freq'].sum()
    prob_no_show = num_no_show/(df['Freq'].sum())
    num_no_show_more_once = df[df['Freq_no_show'] > (num + 1)]['Freq'].sum()
    prob_repeat_no_show = num_no_show_more_once/num_no_show
    return prob_repeat_no_show

4. Conclusion¶

From the above analysis we can see that it is statistically likely that the time between scheduling and the actual appointment may impact the chance of a patient not attending their appointment. Although the statistical resulst were very strong, further data would probably be required to prove this. Also, given the size of the data, the coefficient of determination showed the time between scheduling and the appointment and possibility of a no-show has a small correlation.

We also found that neither gender, being on the scholarship programme, having hipertension, diabetes, alcoholism or receiving a notice sms had any statistical significance on wether a patient attends an appointment or not.

And lastly, we found that for the set of data, a patient who has previously not show up for an appointment, is 18% likely not to attend the following one. This probability increases until 10 'no-shows', upon which it decreases.

This analysis could be useful to the Brazilian hospitals, who could focus their energy on reminding patients who have previously skipped appointments or where there is a long time between scheduling and appointment day.
